  Features
- Multi-page website: Home, Courses, Booking, Portfolio, Contact
- Courses displayed as cards with Book Now buttons
- Booking form saves data in localStorage
- Portfolio images open in a modal when clicked
- Responsive layout
 How to Run
1. Open `index.html` in a browser.
2. Navigate pages using the header links.
3. Book a course through the Booking page to test localStorage.
4. Click portfolio images to view them in modal.
